# 个人博客-Starry

一次全新的尝试，为更好。

## 技术栈：

    前端：nuxt+vuetify
    管理端：vue+elementUi
    服务端：nodeExpress+mongoose

## 功能点：

1.博客发布、删除、修改 、查询 √  
2.图片 oss 对象存储 √  
3.私密博客 √  
4.标签云功能（对博客添加标签，能根据标签查找博客）√  
5.条件查询功能（可以输入条件进行查询 √  
6.评论功能 √  
7.点赞功能 √  
8.邮件通知，全局消息组件 √
